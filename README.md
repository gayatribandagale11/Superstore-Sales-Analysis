📊 FUTURE_DS_01: Business Sales Performance Analytics using Tableau

📌 Project Overview
This project focuses on analyzing business sales data to uncover key trends, identify top-performing products, and evaluate regional performance using Tableau. The goal is to derive actionable insights for better business decision-making.

🛠 Tools Used
- Tableau  
- Data Visualization  
- Data Analysis  

🔍 Analysis Performed
- 📈 Sales trend over time (monthly/yearly analysis)  
- 📦 Category-wise performance comparison  
- 🌍 Region-wise sales analysis  
- 🏆 Top 10 products identification  

📈 Key Insights
- Sales show an overall increasing trend over time  
- Technology category is the highest performing segment  
- West region generates the highest revenue  

💡 Recommendations
- Focus more on high-performing categories like Technology  
- Expand business operations in top-performing regions  
- Promote and market top-selling products to maximize revenue
