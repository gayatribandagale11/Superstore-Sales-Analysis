📊 Sales Dashboard using Tableau
📌 Project Overview
This project focuses on analyzing sales data using Tableau to uncover key business insights.

🔍 Key Insights
- West region contributes the highest sales
- Sales peak in November and December
- February shows lowest performance
- Top products drive major revenue
- Furniture category has strong contribution

  💡 Recommendation
Focus on boosting low-performing months and regions while maximizing revenue from top products.

🛠 Tools Used
- Tableau
- Data Visualization
- Data Analysis
